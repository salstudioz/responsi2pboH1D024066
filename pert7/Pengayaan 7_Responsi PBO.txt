Pertanyaan Refleksi
Setelah menyelesaikan kode, jawab pertanyaan berikut:

1. Apa perbedaan antara interface dengan class biasa?
2. Mengapa interface tidak memiliki implementasi method di dalamnya?
3. Apa tujuan penggunaan keyword "implements" pada sebuah class?
4. Apakah sebuah class dapat mengimplementasikan lebih dari satu interface? Jelaskan dan berikan contoh sederhana.
5. Jika sebuah class mengimplementasikan interface, apa yang terjadi jika class tersebut tidak mengimplementasikan semua method dari interface tersebut?

Jawaban:
1. Perbedaan utama: Interface hanya berisi deklarasi method (tanpa implementasi) dan tidak memiliki constructor, sedangkan class biasa dapat memiliki implementasi method, constructor, dan state. Interface mendukung multiple inheritance (satu class bisa implement banyak interface), sedangkan class hanya bisa extends satu parent class.
2. Interface tidak memiliki implementasi method karena berfungsi sebagai kontrak yang mendefinisikan apa yang harus dilakukan, bukan bagaimana melakukannya. Implementasi diserahkan kepada class yang mengimplementasikan interface tersebut.
3. Keyword "implements" digunakan untuk menunjukkan bahwa sebuah class akan mengimplementasikan semua method yang dideklarasikan dalam interface. Ini adalah kontrak yang harus dipenuhi oleh class tersebut.
4. Ya, sebuah class dapat mengimplementasikan lebih dari satu interface. Contoh: class Smartphone implements Camera, Phone, GPS. Class Smartphone harus mengimplementasikan semua method dari ketiga interface tersebut.
5. Jika sebuah class mengimplementasikan interface tetapi tidak mengimplementasikan semua method dari interface tersebut, maka class tersebut harus dideklarasikan sebagai abstract class. Class tersebut tidak dapat diinstansiasi langsung dan harus di-extends oleh subclass yang mengimplementasikan sisa method tersebut.
